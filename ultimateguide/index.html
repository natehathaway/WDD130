Ultimate Rules

Rules and Regulations. This website will have the rules for playing ultimate “frisbee” according to the UDL, so that people can know how an actual official game is played, how many players on each team, and be able to enjoy the sport better. This will be a great informational page on how it is played, and good practices in the game of Ultimate “frisbee.” People will come to the site because they want to play more but they don’t know how to set up the game, or they can play but they want to get better, learn more throws, or just improve on them.
